## About
I started this project to a college work in UDF realizing the need for bicycle rentals. It's a React Native project made with Expo and Firebase.

## Running
- Make sure you have Expo installed
- Clone project
- `cd mobile`
- `npm install`
- `npm start`
- Enjoy!
